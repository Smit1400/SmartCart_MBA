## Run Locally

Clone the project

```bash
  git clone https://github.com/Smit1400/SmartCart_MBA
```

Go to the project directory

```bash
  cd SmartCart_MBA
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  streamlit run app.y
```
